### Use `cleanAndReshapeImage` programmatically

```python
import imageio
from clean_reshape import cleanAndReshapeImage

img = imageio.imread("/path/to/image.jpg")
processedImg = cleanAndReshapeImage(img)
print(processedImg)
```
